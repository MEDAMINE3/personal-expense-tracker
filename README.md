# 💸 Personal Expense Tracker (Flask + SQLite)

A **FinTech-inspired app** to track personal expenses.  
Built with **Flask**, **SQLite**, **Bootstrap**, and **Chart.js**.

---

## 🚀 Features
- Add expenses (description, amount, category)
- Delete expenses
- See total spent
- Visualize data with charts
